# Practice - Next.js

- Deployed: [NEXT.JS](https://practice-nextjs-eta.vercel.app/)

## Build on local device

```shell
# Change directory
$ cd practice-nodejs

# Required npm v10.*.*
$ npm --version
10.5.2

# Install modules
$ npm install

# Run develop server
$ npm run dev

> next dev
```
